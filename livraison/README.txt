Le dossier de livraison ici présent contient différents dossiers :
Le dossier src contient les sources de l'application,
Le dossier doc la Javadoc compilée,
Le dossier dist contient un sous-dossier bin avec les exécutables Linux et Windows ainsi qu'un sous-dossier lib contenant les archives Jar.
Enfin, le dossier rapport contient le rapport en PDF.
Pour lancer l'application, lancer l'exécutable correspondant à votre système dans une console (ou en cliquant dessus directement).
Pour activer le mode de triche, ajouter n'importe quel caractère comme argument de lancement (sa simple présence va activer le mode).
